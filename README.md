# Disband

A modern, minimalistic desktop chat application inspired by Discord, with enhanced animations, glow effects, and a sleek design.

### Features
- Real-time communication with Socket.IO
- Sleek design using Tailwind CSS
- Electron for cross-platform desktop application support

### Setup
```bash
# Install dependencies
npm install

# Start the application
npm run start
```

### Build for Production
```bash
npm run build
npm run dist
```
